О сайте

Все страницы данного сайта отвечают требованиям специцикации html5 и проходят валидацию в Markup Validation Service. Файл стилей отвечает отвечает требованиям спецификациям CSS2.1 и CSS2 и также проходят валидацию в CSS Validation Service. Сайт также отвечает всем требования, указанным в задании.

Разработка страниц велась в текстовом редакторе SublimeText3. В качестве основы для построения вёрстки основных колонок сайта, использовался генератор css-шаблонов CSStemplater.com.

Тестирование всех страниц сайта производилось в OC Windows в следующих браузерах:

  • Internet Explorer 11, 10, 9 и 8 (IETester и режим совместимости в 11-ом);
  • Mozilla Firefox 29, 28 и 27;
  • Opera 22, 21 и 20;
  • Apple Safari 5.1.7;
  • Google Chrome 33, 32 и 31;

Так как ie8 не отвечает требованиям спецификации html5, во всех версиях ie ниже девятой используется создание некоторой части html-тегов средствами JavaScript. В случае отлючения возможности выполнения скриптов в данных браузерах, в верху страниц выводится сообщение о невозможности корректного отображения в данных браузерах и просьбой включить выполнение скриптов.
