Из менее активных увлечений, найвысший приоритет для меня имеют: написание музыки, макраме, росписью по дереву, резьба по яичной скорлупе, плетение из лозы, вышивание крестиком, вязание крючком, изготовление корабликов в бутылках, конструирование крепостей из спичек и зубочисток, собирание пазлов, разгатывание кроссвордов, аэрография, игра в любительском театре, разведение рыбок, пчеловодство, изучение эльфийского языка, цветоводство и садоводство, астрология, виноделие, коллекционирование бабочек и пивных кружек а также запись сновидений.
