Задание №1

Тема: Программирование форм

  1. Создать три формы для ввода данных. Формы должны содержать следующие элементы: текстовые поля, флажки, переключатели, списки , различные виды кнопок.
  2. При открытии формы предусмотреть установку фокуса на первое поле.
  3. Предусмотреть возможность дополнения и удаления элементов списка.
  4. Форма должна содержать обязательные и необязательные поля. Обязательные поля должны быть выделены.
  5. При вводе элементов формы предусмотреть три варианта проверки данных: проверка вводимых символов, проверка при выходе из элемента управления, проверка при отправке формы.
  6. Введенные данные должны передаваться на сервер. Написать две программы: первая содержит форму и скрипт в разных файлах, вторая – одном. Использовать три способа передачи данных: переменный, индексные массивы, ассоциативные массивы. Переданные данные отображать в виде таблицы.
